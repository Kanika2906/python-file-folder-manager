# 📁 Folder & File Management System

A Python-based command-line application that performs **CRUD (Create, Read, Update, Delete)** operations on both **folders and files**. The project simplifies basic file system management through an easy-to-use menu-driven interface while demonstrating Python's file handling capabilities.

---

## 🚀 Features

### 📂 Folder Management

* Create a new folder
* View existing folders
* Rename folders
* Delete folders

### 📄 File Management

* Create a new file
* Read file contents
* Rename files
* Delete files
* Write data to a file
* Append data to an existing file

### ⚙️ Additional Features

* Menu-driven Command Line Interface (CLI)
* Exception handling for invalid operations
* User-friendly prompts and messages
* Cross-platform support using Python standard libraries

---

```bash

```

3. Run the application

